# Blog Frontend

Next.js 16 frontend for the blog application.

## Development

```bash
npm install
npm run dev
```

The app will be available at `http://localhost:3000`

## Build

```bash
npm run build
npm start
```

## Environment Variables

Create `.env.local`:

```env
NEXT_PUBLIC_API_URL=http://localhost:5000/api
NEXT_PUBLIC_SITE_URL=http://localhost:3000
```
